# Network Communication System Simulation

## Overview  
This project simulates a layered communication system based on the OSI model. It demonstrates data transmission using BPSK modulation over an AWGN channel and evaluates performance using Bit Error Rate (BER).

## Features  
- BPSK modulation and demodulation  
- AWGN channel simulation  
- BER calculation  
- BER vs SNR graph  

## Methodology  
1. Generate random binary data  
2. Modulate using BPSK  
3. Transmit through AWGN channel  
4. Demodulate received signal  
5. Calculate BER for different SNR values  

## Results  
BER decreases as SNR increases, showing improved communication reliability.

## How to Run  
1. Open MATLAB  
2. Run the `.m` file  
3. View BER output and graph
